# Introduction:
This is a project i made that allows you to:
- manage staff and members in a gym
- adding new members/staff
- updating their data
- viewing them
- deleting a member/staff

# Requirements
-python version: 3.11.9
-external packages:
-sqlite3: 3.45.1 
-pandas: 2.3.3
-streamlit: 1.51.0

# once you install or have them installed:
open your terminal in the directory of the python code and then run `streamlit run database_project.py`

and then enjoy :)

# extra notes:
in the side bar there are extra buttons to add 5 sample examples for each members and staffs.
it also allows you to delete the entire database they have no warning so be wary.
